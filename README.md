# how to launch

- add env file:

```bash
PORT = 3001

POSTGRES_USER = admin
POSTGRES_PASSWORD = admin

API_URL = <ip of backend and port>

DB_HOST = db #for docker launch
# DB_HOST = localhost # for local launch

HEADLESS = false # for docker launch
```
